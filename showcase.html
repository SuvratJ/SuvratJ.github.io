Freelance project to create a follicular unit (hair density) analysis application for MacOS. App was made using Qt/C++.

Virtual Linux Filesystem

A personal project to create a virtual linux filesystem using C. The structure of the virtual filesystem is influenced by the ext2 filesystem.

A* maze solver GUI using Python3

A personal project to generate and solve mazes graphically via the A* algorithm. The application was made with Python3 and PyGame.

BoardControl: A Chess visualization web application

A personal project to create a web application using ReactJS and ExpressJS technologies to visualize how control over the squares changes through the course of a chess game.

Pivot hi/lo reversal indicator
