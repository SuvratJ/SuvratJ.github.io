Receipt conversion solution

This is a freelance project I did for a client from Barr Nunn Transportation. They needed an application or script to convert receipts scanned in TIFF to PDF. The requirements were

  • A script or application to convert multipage TIFF files to PDF
  • All TIFF files in a specified direcory should be converted in a single batch

I used Python3 and PIL and decided to implement a command line solution, making it easier for the solution to be used as a sub part of other applications.
