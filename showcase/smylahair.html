SmylaHair: MacOS Application

This is a freelance project I did for a hair transplant clinic. The client wanted a MacOS application for follicular unit analysis, which is essentially hair density measurement for hair transplant. The requirements were

  • Ability to capture images using a USB Microscope
  • Ability to callibrate the microscope with a custom length
  • Ability to mark different types hair follicles on the captured image
  • Ability to mark a custom region for hair density measurement
  • Disply of a summary of densities of all the marked follicles in the custom region

I made the application using Qt framework/C++. Qt is a mature framework and has great libraries for interacting with devices like the USB microscope.
